# Sample Snapshot Data

This directory contains representative snapshot datasets used to reproduce the current-distribution analysis presented in Figure 5 (e-j) of the manuscript.

The files are organized into two example realizations:

```
one_cycle/
```

A realization in which activity terminates after a single cycle.

```
multi_cycle/
```

A realization exhibiting self-sustained multi-cycle activity.

Each directory contains the potential reactivation currents and active-neuron lists required by the notebook

```
figure5_panels_ej.ipynb
```
