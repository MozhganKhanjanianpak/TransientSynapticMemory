# Basic Dynamics

This directory contains the code required to reproduce the **single-realization dynamics** presented in the manuscript.

The workflow is

```
C++ simulation
        ↓
Simulation outputs
        ↓
Python notebooks
        ↓
Figures
```

---

## Directory structure

```
basic_dynamics/

├── cpp/
│
├── python/
│
├── sample_output/
│
└── README.md
```

---

## cpp/

The C++ implementation simulates one realization of the transient synaptic memory model.

The program

- generates a random directed Erdős–Rényi network,
- initializes a random subset of excitatory and inhibitory neurons,
- evolves the network dynamics,
- records activity time series,

All model parameters are defined in

```
parameters.h
```

---

## python/

This directory contains Jupyter notebooks for reproducing the figures based on a **single simulation**.

### `analyze_single_run.ipynb`

Uses the simulation outputs to generate

- activity time series,

corresponding to Figure 2 in the manuscript.

---

## sample_output/

Contains a sample simulation generated by the C++ code.

These files allow users to execute the notebooks immediately without first running the simulation.

Users who wish to generate their own realizations may simply replace these files with outputs from the C++ program.

---

## Typical workflow

```
Compile the C++ code

↓

Run one simulation

↓

Store outputs in sample_output/

↓

Open analyze_single_run.ipynb

↓

Reproduce the figures
```
